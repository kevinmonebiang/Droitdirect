DroitDirect branding source files

- `droitdirect_mark.svg`: compact mark for favicon, splash, app icon export
- `droitdirect_logo.svg`: mark prepared as the main exportable logo source

Recommended exports

- Favicon web: `32x32`, `48x48`
- PWA icon: `192x192`, `512x512`
- Splash / launcher artboard: `1024x1024`
- Social preview: `1200x630`

Base colors

- Navy: `#11284A`
- Navy soft: `#1D3A67`
- Gold: `#C8A96B`
- White: `#FFFFFF`
